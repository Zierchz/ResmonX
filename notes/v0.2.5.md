Unelevated window for ASUS OLED "Target Mode" (and other per-window dimming) compatibility.

## Changes
- The main window now runs unelevated, so per-window dimming features like ASUS OLED Care "Target Mode" keep it lit when focused. All privileged monitoring — ETW per-process network and per-file disk activity, plus process actions — runs in an elevated helper started at launch, over a secured local named pipe. Still a single UAC prompt at startup.

## Install
Download and run `ResmonX-Setup.exe` (or `resmonx_0.2.5_x64-setup.exe`). Requires Windows 10/11.

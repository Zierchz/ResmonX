Floating desktop widget with a system tray.

## Changes
- New always-on-top **floating widget**: a compact, translucent panel with live
  CPU / RAM / GPU / disk / network mini-cards and the full process list
  (sortable, filterable). Click any metric or a row to jump to that section in
  the main window. Drag to move; adjustable opacity; position is remembered.
- **System tray** icon to show/hide the widget, restore the main window, or quit.
- Closing the main window now **minimizes to the tray** instead of exiting;
  "Salir" in the tray is the only full exit. Monitoring keeps running in the
  background.

## Install
Download and run `ResmonX-Setup.exe` (or `resmonx_0.4.0_x64-setup.exe`).
Requires Windows 10/11.

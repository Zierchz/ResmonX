In-app updater: check for and install updates without losing your setup.

## Changes
- **Update button** in the top bar: checks for a newer release on startup and
  on click. When a greater version is published, the icon **pulses**; clicking
  it shows a confirmation with the version and notes, then downloads, installs
  and restarts automatically.
- Updates now install **in place** instead of uninstall + reinstall, so your
  taskbar pin and other shortcuts are preserved.
- Updater artifacts are **signed**; the app verifies the signature before
  installing.

## Notes
- This is the first release with the in-app updater. Update to it once by hand;
  from here on, ResmonX can update itself.

## Install
Download and run `ResmonX-Setup.exe` (or `resmonx_0.5.0_x64-setup.exe`).
Requires Windows 10/11.

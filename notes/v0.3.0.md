Frontend rebuilt on React + Vite + Tailwind CSS + shadcn/ui + TanStack Table.

## Changes
- The whole UI was migrated from vanilla TypeScript to **React** (Vite, Tailwind CSS v4, shadcn/ui, TanStack Table). Same features and dark Task Manager look, now on a component-based, accessible foundation.
- Process tables (sort, filter, per-column heatmap) run on TanStack Table; the heatmap intensity shading now covers every process list (CPU, memory, disk, network, GPU), not just the overview.
- Right-click process menu, confirmation dialogs and toasts rebuilt on Radix primitives, with a single shared menu for smooth performance.
- No backend changes: still a single UAC prompt at startup, the elevated helper for ETW per-process network / per-file disk activity, and the same process actions.

## Install
Download and run `ResmonX-Setup.exe` (or `resmonx_0.3.0_x64-setup.exe`). Requires Windows 10/11.

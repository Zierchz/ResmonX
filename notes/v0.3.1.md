Layout fix for section tables.

## Changes
- Tables on the CPU and Network views no longer overflow the window and force an
  unnecessary page scroll. Each table is now bounded to the available height and
  scrolls internally, capped at the height of the summary cards column.

## Install
Download and run `ResmonX-Setup.exe` (or `resmonx_0.3.1_x64-setup.exe`).
Requires Windows 10/11.
